Welcome!

This web app was created in order to allow users to interact with and explore data from the TESS mission. You can conveniently find data regarding observations, such as which sector is being viewed and by which camera, as well as the observation date.

To get started, simply select which tool you would like to use. From there, you can easily search by entering coordinates, an object name, or TIC ID.
